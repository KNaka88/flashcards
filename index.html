JavaScript and jQuery Flashcards

JavaScript (JS)

Definition:

JavaScript is a programming language used to make web pages interactive.

Operators

Definition:

operators are symbols such as +, -, /, *, %, etc. These are used for mathmatical tasks in javascripts

Variables

Definition:

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

Definition

Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word. Always use the var keyword when initializing variables in JavaScript.

Functions

Definition

A function is a block of code that performs an action and returns a result; optionally takes arguments

Methods

Definition

You can think of a method as an action that a number can perform or have taken on it.

Arguments

Definition

Arguments provide a bit more information to methods to help them know what they're supposed to do. Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

Definition

A parameter is a variable that's assigned to the argument. Above, number1 and number2 are parameters, and 1 and 2 are arguments.

Return

Definition

The return value is the method's response.

Chaining methods

Definition

Calling a method directly on the return value of another method.

Strings

Definition

String is a JavaScript data type that represents the exact text of whatever is enclosed in the quotes

Booleans

Definition

True and false are booleans. They are not strings - they simply represent being true or false.

Undefined

Definition

An object (such as a variable) without a defined value.

NaN

Definition

NAN stands for not a number. It is also a type of number in Javascript.

Escape

Definition

The \\ is called an escape. It tells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string.

Alert

Definition

alert() opens a dialog box and returns undefined

Comments

Definition

anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

jQuery

Definition:

jQuery is a JavaScript library that makes it easier to incorporate JavaScript on your web pages. When we say it is a library, we mean that many common tasks that would take many, many lines of JavaScript code to execute have been packaged into methods that you'll be able to use with a single line of code.

Attributes

Definition:

All HTML elements can have attributes such as title, class, id, width and height, etc. Attributes provide additional information about an element and are always specified in the start tag. Attributes usually come in name/value pairs like: name="value"
